What is CSS?

CSS stands for Cascading Style Sheets
Cascading: Styles flow down and can be overridden (like water cascading down)
Style: The visual appearance of elements (colors, fonts, layout)
Sheets: Files that contain styling rules and instructions

CSS Selectors: Targeting Elements

Learn how to select and style specific HTML elements

How CSS Connects to HTML

CSS works by selecting HTML elements and applying styles to them. Think of it like giving instructions: "Find all headings and make them blue" or "Find this specific paragraph and make it larger."

Basic CSS Structure

CSS Rule Structure 
/* CSS Rule Structure */
selector {
    property: value;
    property: value;
}
selector
Selector

Tells CSS which HTML elements to style

property
Property

What aspect to change (color, size, position)

value
Value

How to change it (red, 16px, center)

Essential CSS Selectors

Element Selector

CSS: h1 { color: blue; }
Result: All <h1> elements become blue

This heading is blue

Class Selector

CSS: .highlight { background: yellow; }
Result: Elements with class="highlight"

This text is highlighted

ID Selector

CSS: #main-title { font-size: 24px; }
Result: Element with id="main-title"

Main Title (Large)

Responsive Design Basics

Making your website look good on all device sizes is crucial. Here are the essentials:

Viewport Meta Tag

Always include this in your HTML head:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Flexible Units

px - Fixed pixels
% - Percentage of parent
rem - Relative to root font size
vw/vh - Viewport width/height

Media Queries

Different styles for different screen sizes:

/* Mobile styles by default */
.container {
    padding: 10px;
}

/* Tablet and larger */
@media (min-width: 768px) {
    .container {
        padding: 20px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        padding: 40px;
    }
}

  • Professional color scheme and spacing
  • Flexbox layout for perfect alignment

Step-by-Step Styling Guide

1

Create External CSS File

Create a new file called profile-styles.css in the same folder as your profile.html

Why external CSS? It keeps our HTML clean and makes styles reusable across multiple pages.
2

Link CSS to HTML

Add this line in the <head> section of your profile.html:

<link rel="stylesheet" href="profile-styles.css">
3

Add Base Styles

Start with these foundational styles in your profile-styles.css:

Base Styles 
/* Reset and base styles */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
    line-height: 1.6;
    color: #333;
    background-color: #f8f9ff;
}

.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 20px;
}
4

Style the Header

Make your header stunning with gradients and typography:

Header Styles 
/* Header styling */
header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    color: white;
    padding: 60px 0;
    text-align: center;
    position: relative;
    overflow: hidden;
}

header::before {
       content: '';
       position: absolute;
       top: 0;
       left: 0;
       right: 0;
       bottom: 0;
       background: rgba(255, 255, 255, 0.05);
}

header .container {
    position: relative;
    z-index: 2;
}

header h1 {
    font-size: 3rem;
    font-weight: 700;
    margin-bottom: 15px;
    text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

header p {
    font-size: 1.2rem;
    font-weight: 300;
    opacity: 0.95;
    max-width: 600px;
    margin: 0 auto;
}
5

Create Beautiful Navigation

Style your navigation with hover effects:

Navigation Styles 
/* Navigation styling */
nav {
    background: white;
    padding: 20px 0;

    top: 0;

    flex-wrap: wrap;

    text-decoration: none;
    color: #333;
    font-weight: 600;
    padding: 10px 20px;
    border-radius: 25px;

    transition: width 0.3s ease;
}

    width: 80%;
}
